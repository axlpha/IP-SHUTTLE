# IP Shuttle

## Quick Start

```bash
# Download the JAR
wget https://github.com/axlpha/ipshuttle/releases/latest/ipshuttle.jar

# Run it
java -jar ipshuttle.jar

# Set your browser proxy to:
socks5://localhost:43858
```

## Usage

1. Launch the app: `java -jar ipshuttle.jar`
2. Click **Select Proxy** and choose one from the list.
3. Click the big **ON** button to start the local SOCKS5 server on `localhost:43858`.
4. Point your browser or application to `socks5://localhost:43858`.

That's it – your traffic is now routed through a fresh residential IP.
